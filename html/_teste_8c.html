int regGastos ( char * pdt,
int cate_g,
float preco,
int qtde_g )

Realiza o registro de gastos com base nos dados de compras.

Parameters
pdtNome do produto ou item comprado.
cate_gCategoria do gasto (1 para Roupas, 2 para Cosmeticos).
precoPreco unitario do produto (deve ser maior ou igual a zero).
qtde_gQuantidade comprada (deve ser um valor inteiro maior que zero).
Returns
int Retorna 1 se o registro for valido, retorna 0 se um ou mais parametros forem invalidos.
Note
  • O parametro cate_g aceita apenas os valores 1 (Roupa) ou 2 (Cosmetico).
  • O parametro preco aceita numeros decimais e inteiros positivos ou zero. Valores negativos tornam o gasto invalido.
  • O parametro qtde_g aceita apenas inteiros maiores que zero. Valores menores ou iguais a zero tornam o gasto invalido.
